### 연습문제
2주차 연습문제 맵핑과 컨트롤러 추가하기 완료.
-HTTP 요청을 처리하는 컨트롤러와 매핑 구조를 직접 구성해 동작 흐름을 확인

3주차 연습문제 상세 페이지 수정하기 완료.

4주차 연습문제 사용자 추가 및 출력하기 완료.

5주차 연습문제 페이지 리다이렉트 완료.
-요청 처리 후 다른 URL로 흐름을 전환하는 리다이렉트 동작 방식을 사용
